# Mini Games Hub

A collection of simple browser-based games built using HTML, CSS, and JavaScript.

This project includes multiple interactive games designed to practice JavaScript logic, DOM manipulation, and basic game development concepts.


## Games Included

### Memory Game
Flip the cards and match all pairs to win.

### Number Guessing Game
Guess the correct number within a given range.

### Snake Game
Control the snake using arrow keys and try to survive while eating food.

### Car Dodge Game
Move your car left/right and avoid incoming obstacles.


## Rules

- Each game has a **Start button** to begin.
- Follow on-screen instructions for controls.
- Winning and losing messages are displayed after each game.
- Refresh or restart to play again.
